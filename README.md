## Tasks CLI

Tasks CLI is a program to manage your tasks in a database using terminal or console. This is a sample project for beginners

### Requirements

- Nodejs
- Mongodb Local Installation or Cloud

### Installation

```sh
npm install
```

```sh
npm link
```

### Commands

```
tasks-cli --version
```

```
tasks-cli --help
```

```
tasks-cli list
tasks-cli l
```

```
tasks-cli find title
```

```sh
tasks-cli save
tasks-cli s
```

```sh
tasks-cli update <id>
```

```sh
tasks-cli delete <id>
tasks-cli d <id>
```
